## Getting Started

```bash
# Cloning the repository
git clone https://github.com/trickvg/podcastr.git

cd podcastr
```

### Install dependencies and running application

##### Yarn

```bash
#Installing dependencies
yarn install

# Running app
yarn dev

# Starting json server
yarn server
```
##### NPM
```bash
#Installing dependencies
npm install

# Running app
npm run dev

# Starting json server
npm server
```
